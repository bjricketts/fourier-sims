# Fourier-sims

This repository contains the python scripts for generating simulated light curves to show the effects of 
calculating the cross-spectrum between two light curves of differing statistical properties (different 
power spectra). Figures generated by this script were used in X-ray timing signals and phase lags in 
X-ray binaries I. Simple simulated signals produce non-linear phase lags (Ricketts and Marcel, 2026). 

## Requirements
- numpy
- stingray
- matplotlib
- ndspec
- scipy

## How to generate paper plots

All plots are generated via a CLI interface with QPO sims. There are models for each plot/model in the
paper which are selected from the following models:

- damping
- coherent_sum
- resonance
- hidden
- multi_incoherent
- multi_coherent
- multi_multiplicative
- fm
- count_rate_fm
- enevelope_qpo

By default, figures are outputted into a figures subfolder. Thus to generate the figures seen in the 
paper, one should perform the following commands:

```
mkdir -p figures
python qpo_sims.py damping          --outdir figures --seed 0   # Fig. 1
python qpo_sims.py coherent_sum     --outdir figures --seed 0   # Fig. 2
python qpo_sims.py hidden           --outdir figures --seed 0   # Fig. 4
python qpo_sims.py multi_incoherent --outdir figures --seed 0   # Fig. 6 (a)
python qpo_sims.py multi_coherent   --outdir figures --seed 0   # Fig. 6 (b)
```

All models default to the parameters used in the paper.
